# Simulation eines Ameisenhaufens

## Funktionalität
Stufe 1:
- flaches Gelände
- 1 Haufen mit n Ameisen
- n Nahrungsquellen (evt. abstrakt, kann auch konkret sein z.B. Apfel)
- Hauptbeschäftigung: Futtersuche -> wenn eine Ameise etwas findet markiert sie den Weg, die anderen folgen dem Weg. Wenn Nahrung verbraucht "erlischt" der Weg

Erweiterungen:
- variables Gelände, Hindernisse (Wasser)
- verschiedene Gattungen (Königin, Arbeiter, Kämpfer, ...)
- Ameisen sterben und vermehren sich
- mehrere Ameisenhaufen -> Kampf um Nahrung -> neue Gattung Krieger

## Technik
Basis:
- Server in Java
- REST API
- Client in JavaFX
- git
- maven

Erweiterungen:
- Persistenz
- Multiplayer
- Client im Browser/JavaScript (auf dem Server gehostet)
